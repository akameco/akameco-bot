akameco bot

@akamecoに実装されているBOTの機能集。 個々の機能はBOT作成モジュールfoxtailを使用。

update_name

改名する 
        
export default (fox) => {
  fox.add((res) => {
    if(/.*\(@akameco\)$/.test(res.text)) {
      let name = res.text.match(/(.*)\(@akameco\)/)[1]
      if(name.length > 20) {
        res.reply('20文字以上の名前は設定不可能なんっ！')
        return
      }
      fox.twit.post('account/update_profile', {name: name}, () => {
        res.reply(`${name}に改名しました`)
      })
    }
  })
}

おるかｗ

"あかめ"、"赤目"、"赤芽"＋"おるか"に「おるでｗ」と反応。それだけ。 
      
export default (fox) => {
  fox.add((res) => {
    if(/^(赤目|あかめ|赤芽)おるか/.test(res.text)) {
      res.reply('おるでw')
    }
  })
}
