## Usage

### Set keys

Set your keys.

```
$ export CONSUMER_KEY= ...
$ export CONSUMER_SECRET= ...
$ export ACCESS_TOKEN= ...
$ export ACCESS_TOKEN_SECRET= ...
```

### npm start

node index.js

```
$ npm start
```

## Development

```
# node index.js
npm start
# watch
npm run watch
# build
npm run build
```
